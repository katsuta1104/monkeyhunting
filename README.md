## 使い方
<ol>
  <li>コードをコピーする
</li>
  <li>colabに貼って実行する
  </li>
  <li>指示に従う
  </li>
</ol>


## 機能
<il>
  <li>
    初速度、距離、高さ、投射角(仰角)を設定できる
  </li>
  <li>自動的に縮尺が調整される
</li>
  <li>
    弾丸の向きが線として表示される
  </li>
  <li>
    アニメーションとして直感的に理解できる
  </li>
</il>
